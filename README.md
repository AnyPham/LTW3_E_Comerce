# LTW3_E_Comerce
# 🛒 Dự Án Website Bán Hàng Fullstack

> **Công Nghệ Sử Dụng**  
> 🟢 React JS, MongoDB, Express JS, Node JS, Material UI

---

## 🚀 Tính Năng Nổi Bật

- Giao diện hiện đại với **Material UI**
- **Frontend** responsive sử dụng **React JS**
- **Backend** mạnh mẽ với **Node.js** & **Express.js**
- Cơ sở dữ liệu linh hoạt với **MongoDB**
- Kiến trúc Fullstack sẵn sàng **triển khai**

---

## 📦 Công Nghệ

| Frontend  | Backend     | Cơ sở dữ liệu | Thiết kế giao diện |
|----------|------------|---------------|--------------------|
| React JS | Node.js     | MongoDB       | Material UI        |
| Redux    | Express.js  |               | Responsive         |
